# Laravel Relationships Practice

This repository is dedicated to mastering Laravel Eloquent Relationships through practical, project-based learning. Instead of relying on database seeders, I build small applications where data is created through forms, just like in real-world projects.

## 🎯 Goal

The goal of this repository is to understand every Laravel relationship by implementing it in practical CRUD projects.

## 💡 Learning Approach

This repository focuses on learning Laravel by building practical applications.

* No database seeders for the main projects
* Data is created and managed through forms
* Every relationship is implemented in a real CRUD workflow
* Each project introduces a new relationship concept and builds on previous knowledge

## 📖 What I Have Learned

* Database design for relationships
* Foreign keys and migrations
* Eloquent relationship methods
* Eager loading
* Pivot tables
* Relationship CRUD operations
* Managing many-to-many relationships with `attach()`, `detach()`, and `sync()`
